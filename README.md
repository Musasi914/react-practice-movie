# Movie App

## Description

React を久しぶりに触ったので、その練習として作成したもの。

API は TMDB を使用。

## Tech Stack

- React
- TypeScript
- Tailwind CSS
- Shadcn UI

## Features

- Search for movies
- View movie details
- View movie trailers
- Filter movies by year and rating
- Sort movies by popularity, vote average, and release date
- Pagination
- Favorite movies
- Dark mode
- Responsive design
- TypeScript
- React
- Tailwind CSS
- Shadcn UI

## Usage

```bash
npm install
npm run dev
```
